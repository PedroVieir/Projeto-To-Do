# ToDo+ Frontend (Vite + React + Tailwind)

Compatível com seu backend ToDo+:
- Base da API via `VITE_API_BASE` (ex.: `http://localhost:5050/api`)

## Rodar local

```bash
cd todo-plus-frontend-v5
npm i
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Notas
- Tema claro/escuro com persistência em `localStorage`.
- Dashboard com confete ao concluir tarefa.
- Criação de tarefa pessoal ou de grupo (se houver grupos).
- Páginas: Login, Register, Dashboard, Groups.
